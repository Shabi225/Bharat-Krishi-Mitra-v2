Farmer Scheme Information Portal - A multilingual web application providing Indian farmers access to government agricultural schemes with voice interaction support.

Bharat Krishi Mitra is a full-stack web application designed to empower Indian farmers by providing easy access to central and state government agricultural schemes. Built with modern web technologies, it features a responsive UI, multilingual support (Hindi/English), and voice-enabled navigation for low-literacy users.

Core Features:
- Comprehensive database of agricultural government schemes
- Multilingual interface (Hindi + English)
- Responsive design for mobile/desktop
- Voice interaction (Text-to-Speech, Speech-to-Text) - **Future**
- Offline PWA support - **Planned**
